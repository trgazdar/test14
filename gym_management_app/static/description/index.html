Odoo Gym Management System Apps

Gym Management odoo apps helps user to manage gym members, gym trainers, workout plan, diet plan, bmi, weight, exercise overview. Also using this module users can buy gym membership, users can see analysis of members with invoiced and report for the workout schedule and diet schedule.

Gym Members, Workout Plan, Diet Plan, BMI & Weight.

Users can see the Members form. And also users can see the it's workout plan, diet plan, bmi & weight. .

Gym Trainers

Users can see the Trainers form. and also user can see the it's public Information, personal information and HR settings.

Exercise Overview & Workout Plan

Users can see the workout plan & list of exercises.

Diet Food and Diet Plan

Display diet food, diet plan and also display diet interval.

Diet Schedule PDF Report & Workout Schedule PDF Report

User can print diet schedule pdf report & workout schedule pdf report containing diet plan and workout plan.

Member Analysis.

User can see member pivot analysis and different graph like bar graph, line chart, and pie chart.
